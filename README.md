 
# PDF → Texto (Client-Side)

Ferramenta web para extrair texto de arquivos PDF diretamente no navegador.  
Sem backend. Sem upload para servidor. Sem dependências externas além do PDF.js.

---

## Objetivo

Disponibilizar uma solução simples e funcional para converter PDFs em texto editável, executando 100% no client-side, compatível com GitHub Pages.

---

## Funcionalidades

- Upload de arquivo PDF
- Extração automática de texto
- Contador de páginas
- Barra de progresso
- Visualização do texto extraído
- Download do resultado em arquivo .txt

---

## Tecnologias

- HTML5  
- CSS3  
- JavaScript (ES6+)  
- PDF.js (Mozilla)

---

## Estrutura
pdf-to-text/
├── index.html
├── style.css
└── script.js
---

## Execução

Basta abrir o `index.html` em qualquer navegador moderno ou acessar a versão publicada no GitHub Pages https://alexsandrosipelli.github.io/PDF-Texto-Conversor-Online/.
 
---

## Observações Técnicas

- Todo processamento ocorre no navegador
- Nenhum arquivo é enviado para servidor
- Compatível com navegadores modernos (Chrome, Edge, Firefox)
